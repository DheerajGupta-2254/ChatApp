# ChatApp 💬

A full-stack real-time chat application built using MERN stack and Socket.IO.

## 🔧 Features

- Real-time messaging
- User login/signup
- Group and private chats
- Online/offline indicators

## 🛠 Tech Stack

- React.js (Frontend)
- Node.js + Express (Backend)
- MongoDB (Database)
- Socket.IO (WebSocket)
